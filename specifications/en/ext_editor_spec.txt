ExtEditor FUNCTIONALITY REQUIREMENTS

ExtEditor(QTextEdit) — extended text editor with HTML syntax highlighting
and tag auto-completion.

CLASS HtmlHighlighter(QSyntaxHighlighter):
  - Highlights HTML tags (<tagname>) in blue, bold font.
  - Highlights single and double quoted content in green.
  - Highlights HTML comments (<!-- -->) in red.
  - Rules are defined via QRegularExpression, applied in highlightBlock().

CLASS ExtEditor(QTextEdit):
  - Creates HtmlHighlighter for syntax highlighting.
  - autocompletion_tags: list of known tags (p, b, div, span, a, ul, li,
    h1–h6, table, tr, td) for which auto-completion is performed.

  keyPressEvent():
    - When > is typed, searches for the last < before cursor.
    - If the text between < and > matches a known (non-closing) tag,
      automatically inserts the closing tag and places cursor between them.
    - Otherwise — standard processing.
