FileEditor FUNCTIONALITY REQUIREMENTS

1. FileEditor is a QWidget that wraps a QTextEdit (or compatible subclass
   such as ExtEditor). It performs file operations for a single document:
   loads from disk, saves to disk, tracks modification state, and detects
   external changes to the underlying file.

2. CLASS: FileEditor(QWidget)
   - Constructor: __init__(parent=None, editor_class=QTextEdit, path=None)
   - editor_class allows dependency injection of any QTextEdit subclass.
   - path — pathlib.Path for an existing file or None (new unsaved file).

3. CLASS ATTRIBUTES
   - untitled_name = "untitled" — name for new unsaved files.
   - modification_label = "*" — marker indicating modified document.

4. SIGNALS
   - user_modification_changed = pyqtSignal(str, str, str)
     Parameters: (file_name, full_path, modification_label)
     Emitted when the document is modified by the user (not programmatically).
   - line_changed = pyqtSignal(int)
     Parameter: line number (1-based).
   - message = pyqtSignal(str, str, str)
     Parameters: (message_text, sender, message_type)
     message_type: 'debug' | 'info' | 'warning' | 'error'

5. INSTANCE ATTRIBUTES
   - self.editor: text editor widget (QTextEdit or subclass).
   - self.path: pathlib.Path or None.
   - self.is_programmatic_modification_change: bool — programmatic change flag.
   - self.last_file_mtime: datetime.datetime — timestamp of last save/load.

6. METHODS

   6.1. Document state:
        - isModified() -> bool: returns self.editor.document().isModified()
        - setModified(flag): self.editor.document().setModified(flag)

   6.2. File info:
        - get_path_name() -> str: path.name or untitled_name
        - get_full_path() -> str: str(path) or untitled_name
        - get_modification_label() -> str: '*' if isModified(), else ''
        - get_info() -> tuple: (get_path_name(), get_full_path(), get_modification_label())

   6.3. Signaling:
        - emit_user_modification_changed(): gathers info and emits user_modification_changed
        - on_editor_modification_changed(flag): if not is_programmatic_modification_change,
          calls emit_user_modification_changed()
        - on_cursor_position_changed(): emits line_changed with line number (1-based)

   6.4. File operations:
        - load_from_file() -> bool: reads file from self.path, sets content
          in editor. On error emits new_message with type 'error', returns False.
          Uses is_programmatic_modification_change to avoid triggering user_modification_changed.
        - save_to_file() -> bool: writes editor content to self.path.
          Resets modified flag without emitting user_modification_changed.
          On error emits new_message with type 'error'.
        - save_as_file(path) -> bool: sets self.path = path and calls save_to_file().

   6.5. External change detection:
        - is_externally_modified() -> bool: compares last_file_mtime with file mtime on disk.
          If the file is newer — returns True.
