 ТРЕБОВАНИЯ К ФУНКЦИОНАЛЬНОСТИ  FileEditor

1. FileEditor - это QWidget, который обертывает QTextEdit (или совместимый с ним
   подкласс, такой как ExtEditor). Выполняет файловые функции для отдельного документа:
   загружает его с диска, сохраняет на диск, отслеживает состояние изменений и обнаруживает
   внешние изменения в базовом файле.

2. КЛАСС: FileEditor(QWidget)
   - Конструктор: __init__(parent=None, editor_class=QTextEdit, path=None)
   - editor_class разрешает внедрение зависимостей любого подкласса QTextEdit.
   - path — pathlib.Path для существующего файла или None (новый несохранённый файл).

3. АТРИБУТЫ КЛАССА
   - untitled_name = "untitled" — имя для новых несохранённых файлов.
   - modification_label = "*" — метка, указывающая на изменённый документ.

4. СИГНАЛЫ
   - user_modification_changed = pyqtSignal(str, str, str)
     Параметры: (file_name, full_path, modification_label)
     Излучается при изменении документа пользователем (не программно).
   - line_changed = pyqtSignal(int)
     Параметр: номер строки (1-based).
   - message = pyqtSignal(str, str, str)
     Параметры: (message_text, sender, message_type)
     message_type: 'debug' | 'info' | 'warning' | 'error'

5. АТРИБУТЫ ЭКЗЕМПЛЯРА
   - self.editor: виджет текстового редактора (QTextEdit или подкласс).
   - self.path: pathlib.Path или None.
   - self.is_programmatic_modification_change: bool — флаг программного изменения.
   - self.last_file_mtime: datetime.datetime — время последнего сохранения/загрузки.

6. МЕТОДЫ

   6.1. Состояние документа:
        - isModified() -> bool: возвращает self.editor.document().isModified()
        - setModified(flag): self.editor.document().setModified(flag)

   6.2. Информация о файле:
        - get_path_name() -> str: path.name или untitled_name
        - get_full_path() -> str: str(path) или untitled_name
        - get_modification_label() -> str: '*' если isModified(), иначе ''
        - get_info() -> tuple: (get_path_name(), get_full_path(), get_modification_label())

   6.3. Сигнализация:
        - emit_user_modification_changed(): собирает info и излучает user_modification_changed
        - on_editor_modification_changed(flag): если не is_programmatic_modification_change,
          вызывает emit_user_modification_changed()
        - on_cursor_position_changed(): излучает line_changed с номером строки (1-based)

   6.4. Файловые операции:
        - load_from_file() -> bool: читает файл из self.path, устанавливает содержимое
          в editor. При ошибке излучает new_message с типом 'error', возвращает False.
          Использует is_programmatic_modification_change, чтобы не вызвать user_modification_changed.
        - save_to_file() -> bool: записывает содержимое editor в self.path.
          Сбрасывает флаг modified без излучения user_modification_changed.
          При ошибке излучает new_message с типом 'error'.
        - save_as_file(path) -> bool: устанавливает self.path = path и вызывает save_to_file().

   6.5. Проверка внешних изменений:
        - is_externally_modified() -> bool: сравнивает last_file_mtime с mtime файла на диске.
          Если файл новее — возвращает True.
