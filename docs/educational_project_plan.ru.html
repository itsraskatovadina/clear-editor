📚 План учебного проекта: ClearEditor

Эволюция архитектуры ПО от "простого" к "правильному"

О проекте: ClearEditor — учебный многостраничный текстовый редактор на Python + PyQt5. Проект создан для демонстрации ключевых принципов архитектуры ПО и эволюционного рефакторинга.

Цель для студентов: пройти путь от "работающего, но запутанного" кода к "чистой, тестируемой и расширяемой" архитектуре.

Часть 1. Текущая архитектура ("Было")

📁 Структура проекта

Часть 2. Целевая архитектура ("Стало")

🎨 Основные принципы новой архитектуры

  • Разделение на слои (Layered Architecture) — Presentation Layer (GUI), Business Logic Layer (Services), Data Layer (Models)
  • Внедрение зависимостей (Dependency Injection) — компоненты получают сервисы через конструктор, а не создают сами
  • Единая ответственность (SRP) — каждый класс делает только одну вещь
  • Тестируемость — бизнес-логика тестируется без PyQt

📁 Новая структура

clear-editor/
├── core/              # Presentation Layer (GUI)
│   ├── main_window.py    # Только координация

Часть 4. Ключевые изменения в коде (примеры)

Пример 1: Сохранение файла

Было (core/file_edit.py):

def save_file(self):
    try:
        with open(self.file_path, 'w', encoding='utf-8') as f:
            f.write(self.toPlainText())
        self.document().setModified(False)
    except Exception as e:
        QMessageBox.critical(self, "Ошибка", f"Не удалось сохранить: {e}")

Стало (services/file_service.py + core/file_edit.py):

# В сервисе — только бизнес-логика
class FileService:
    def save_file(self, state: FileState) -> None:
        try:
            with open(state.file_path, 'w', encoding=state.encoding) as f:
                f.write(state.content)
            state.is_modified = False
        except Exception as e:
            raise IOError(f"Ошибка сохранения: {e}")

# Во вьюхе — только координация
def save_current_tab(self):
    current_edit = self.tab_panel.current_widget()
    if current_edit:
        state = current_edit.get_state()
        try:
            self.file_service.save_file(state)
            current_edit.set_state(state)
        except IOError as e:
            self.msg_panel.show_error(str(e))

Пример 2: Загрузка сессии

Было (core/editor_app.py):

def load_session(self):
    with open('config.json', 'r') as f:
        config = json.load(f)
    for file_path in config.get('open_files', []):
        if os.path.exists(file_path):
            self.open_file(file_path)

Стало (services/session_service.py + core/main_window.py):

# В сервисе
class SessionService:
    def load_session(self) -> List[Path]:
        config = self.settings_service.get_config()
        return [Path(p) for p in config.get('open_files', []) 
                if Path(p).exists()]

# В главном окне
def _restore_session(self):
    paths = self.session_service.load_session()
    for path in paths:
        self._on_tab_open_request(path)
