ПОСЛЕ РЕФАКТОРИНГА
   FileTabView - чистый QTabWidget: ( вкладки, их тексты/тултипы, сигнализирует о переключении/закрытии).
   EditorWidget - чистый QWidget: ( QTextEdit, сигнализирует об изменениях, без файловой логики).
   FileTabSrv - вся логика: open/save/close/reload, сессия, диалоги, внешние изменения;
   владеет состоянием: какие файлы открыты, какой текущий; сохранение/загрузка/закрытие; внешние изменения; восстановление сессии;
   FileTabSrv подключается к сигналам View, вызывает методы View для обновления  UI
